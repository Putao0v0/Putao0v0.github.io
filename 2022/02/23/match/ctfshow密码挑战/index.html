# 真・biginner

给出了 m<<10000 十进制的后 175 位。
化为数学公式就是
m*2^10000 %10^175=c
m= c*2^-10000 mod 10^175
m= c*2^-10000 mod 5^175

# ^o^

我们知道
m^o^(2^2^_ mod n)=c
已知 m o _ c
就是只要求
2^2^_ mod n
这里好像因为数据太大算不出来
看了大佬的 wp
设2^_=x*phi+k
2^(x*phi+k) mod n
2^(x*phi)*2^k mod n
根据欧拉定理 2^phi mod n =1
所以上式变为 2^k mod n 而 k=2^_ mod phi
